# ⚡ Energy Consumption & Sustainability Analysis (Power BI)
## 📊 Project Overview
This project analyzes energy consumption patterns, renewable energy share, CO2 emissions, and peak demand across different states in India using Power BI.
## 📁 Files Included
* Power BI Dashboard (.pbix)
* Dataset (.csv)
* Dashboard Screenshots
## 📌 Key Insights
* Residential sector consumes the highest energy
* Tamil Nadu shows higher renewable share
* CO2 emissions are directly linked with energy consumption
* Peak demand varies across districts
## 🛠 Tools Used
* Power BI
* Excel / CSV Dataset
